Kernel

Kernel

Let \(T: V \to W\) be a Linear Transformation between s. The kernel (or null space) of \(T\) is the set of all vectors in \(V\) that map to the zero vector in \(W\):

\[\ker(T) = \{\mathbf{v} \in V : T(\mathbf{v}) = \mathbf{0}_W\}\]

Alternative Names

The kernel is also known as: - Null space (denoted \(\text{null}(T)\) or \(N(T)\)) - The pre-image of zero: \(T^{-1}(\{\mathbf{0}_W\})\)

Properties

  1. Subspace: \(\ker(T)\) is always a subspace of \(V\)

    • Contains \(\mathbf{0}_V\) since \(T(\mathbf{0}_V) = \mathbf{0}_W\)
    • Closed under addition: if \(\mathbf{u}, \mathbf{v} \in \ker(T)\), then \(T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v}) = \mathbf{0} + \mathbf{0} = \mathbf{0}\)
    • Closed under scalar multiplication: if \(\mathbf{v} \in \ker(T)\) and \(a \in F\), then \(T(a\mathbf{v}) = aT(\mathbf{v}) = a\mathbf{0} = \mathbf{0}\)

  2. Injectivity criterion: \(T\) is injective (one-to-one) if and only if \(\ker(T) = \{\mathbf{0}_V\}\). In categorical terms, this means \(T\) is a monomorphism

  3. Dimension: The dimension of \(\ker(T)\) is called the nullity of \(T\)

Example

For a matrix transformation \(T_A: \mathbb{R}^n \to \mathbb{R}^m\) defined by \(T_A(\mathbf{x}) = A\mathbf{x}\): \[\ker(T_A) = \{\mathbf{x} \in \mathbb{R}^n : A\mathbf{x} = \mathbf{0}\}\]

This is precisely the solution set of the homogeneous linear system \(A\mathbf{x} = \mathbf{0}\).

Kernel in Group Theory

The concept of kernel extends to Group Homomorphism between groups. If \(\phi: G \to H\) is a group homomorphism, then:

\[\ker(\phi) = \{g \in G : \phi(g) = e_H\}\]

where \(e_H\) is the identity element of \(H\).

Important: The kernel of a group homomorphism is always a Normal Subgroup of the domain. This is because for any \(g \in G\) and \(k \in \ker(\phi)\):

\[\phi(gkg^{-1}) = \phi(g)\phi(k)\phi(g^{-1}) = \phi(g)e_H\phi(g)^{-1} = \phi(g)\phi(g)^{-1} = e_H\]

Therefore \(gkg^{-1} \in \ker(\phi)\), showing that \(\ker(\phi)\) is invariant under conjugation.

Quotient Groups and the First Isomorphism Theorem

Since the kernel is a normal subgroup, we can form the Quotient Group \(G/\ker(\phi)\). The First Isomorphism Theorem states that:

\[G/\ker(\phi) \cong \text{Im}(\phi)\]

This fundamental result shows that every homomorphic image is isomorphic to a quotient group. See First Isomorphism Theorem for the complete statement and proof.

Kernel in Ring Theory

For ring homomorphisms (maps that preserve both addition and multiplication), the kernel takes on additional structure. If \(\phi: R \to S\) is a ring homomorphism, then:

\[\ker(\phi) = \{r \in R : \phi(r) = 0_S\}\]

The kernel of a ring homomorphism is always an Ideal of the domain ring \(R\). This is the ring-theoretic analogue of the fact that kernels of group homomorphisms are normal subgroups. Just as normal subgroups allow us to form quotient groups, ideals allow us to form quotient rings.
