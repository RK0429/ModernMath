Order

Order

In group theory, the term order has two related but distinct meanings:

Order of a Group

The order of a group \(G\), denoted \(|G|\) or \(\text{ord}(G)\), is the cardinality of \(G\): - For finite groups: the number of elements in \(G\) - For infinite groups: said to be infinite

Examples

  • \(|\mathbb{Z}_n| = n\) for the cyclic group of integers modulo \(n\)
  • \(|S_n| = n!\) for the symmetric group on \(n\) elements
  • \(|\mathbb{Z}| = \infty\) for the group of integers under addition

Order of an Element

The order of an element \(g\) in a group \(G\), denoted \(|g|\) or \(\text{ord}(g)\), is the smallest positive integer \(n\) such that: \[g^n = e\] where \(e\) is the identity element of \(G\).

If no such positive integer exists, the element has infinite order.

Equivalent Definition

The order of an element \(g\) equals the order of the cyclic subgroup generated by \(g\): \[|g| = |\langle g \rangle|\]

Examples

  • In \(\mathbb{Z}_6\): \(|2| = 3\) since \(2 + 2 + 2 = 6 \equiv 0 \pmod{6}\)
  • In \((\mathbb{Z}, +)\): every non-zero element has infinite order
  • In any group: \(|e| = 1\) (only the identity has order 1)

Properties

For Groups

  1. A group has order 1 if and only if it is the trivial group \(\{e\}\)
  2. Groups of prime order are cyclic
  3. The order of a subgroup divides the order of the group (see Lagrange‚Äôs Theorem)

For Elements

  1. \(g^{|g|} = e\) and \(|g|\) is the smallest such positive integer
  2. \(g^k = e\) if and only if \(|g|\) divides \(k\)
  3. \(|g^{-1}| = |g|\) (an element and its inverse have the same order)
  4. If \(g\) and \(h\) commute, then \(|gh|\) divides \(\text{lcm}(|g|, |h|)\) (see @../number-theory/def-lcm)
  5. Conjugate elements have the same order: \(|hgh^{-1}| = |g|\)

Connection to Cyclic Groups

An element \(g\) has order \(n\) if and only if the cyclic subgroup \(\langle g \rangle\) it generates has exactly \(n\) elements: \[\langle g \rangle = \{e, g, g^2, \ldots, g^{n-1}\}\]

A group \(G\) is cyclic if and only if it contains an element whose order equals \(|G|\).
